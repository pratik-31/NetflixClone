# NETFLIX CLONE

This project is a simply front end clone of Netflix. It was created with React and Tailwind. It uses The MovieDB Api to search for movies and display details.

## PACKAGES I USE

* React-youtube
* TMDB Api
* Axios
* Movie Trailer
* TailwindCss
